# Shortest Public Quickstart Snippet

Build a deterministic offline valuation scenario packet from fictional local fixtures.

## Quickstart

- `python -m venv .venv`
- `. .venv/bin/activate`
- `pip install -e .`
- `valuation-scenario-lab demo`
- `valuation-scenario-lab selfcheck --root .`

## Artifact Map

- `demo/valuation-packet.html`: primary packet (ok)
- `demo/showcase-dashboard.html`: visual summary (ok)
- `demo/thesis-brief.html`: analyst narrative (ok)
- `demo/scenario-library.html`: assumption cards (ok)
- `demo/casebook.html`: stranger-readable walkthrough (ok)
- `demo/reviewer-scorecard.html`: reviewer operability (ok)
- `demo/troubleshoot.html`: diagnostic map (ok)
- `demo/reproducibility-audit.html`: repeatability receipt (ok)
- `release/public-bundle.html`: public bundle index (ok)

## Boundaries

- No live data.
- No broker connections.
- No buy/sell/hold advice.
